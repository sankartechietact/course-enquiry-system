# Course Enquiry Management System

A Spring Boot based web application that allows users to submit course enquiries
and enables admins to manage courses and view enquiries through an admin panel.

## Features
- User course enquiry form
- Admin login & dashboard
- Add / update / delete courses
- View user enquiries
- Email notification support
- Responsive UI using Bootstrap & Thymeleaf

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- MySQL
- Bootstrap
- Maven

## Project Structure
- Controller layer
- Service layer
- Repository layer
- Entity models
- Thymeleaf templates
